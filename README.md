## 1. 개 요
* * *
이번 프로젝트에서는 이미지 검색 인공지능 API를 활용해서
 이미지 검색 서비스를 제공해보고자 한다. 가구를 사진으로 찍거나
 인터넷에 있는 가구사진을 이미지로 검색하여 쇼핑몰과 연계해
 쇼핑에 도움이 되는 서비스를 제공한다.
